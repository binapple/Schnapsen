## Build Process
This project uses **Gradle (Kotlin DSL)**.
1. Clone the repository.
2. Open the project e.g. in **IntelliJ IDEA** (Configuration files included in `.idea/`).
3. Run `./gradlew build` to compile.

## 3rd-Party Software
